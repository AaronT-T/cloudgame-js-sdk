## TCGSDK 简介
腾讯云云游戏JSSDK，用于web接入，支持端游和手游。SDK提供了丰富的接口，满足大部分接入需求。

生命周期，调用方式，详细接口描述请参考官网[JSSDK](https://cloud.tencent.com/document/product/1162/46134)。
## 安装

直接 import dist/tcg-sdk，导出的为umd 格式
